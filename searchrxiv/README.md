# searchRxiv Submission Packet

This folder contains a manual-upload packet for depositing the executed search strategies at `searchRxiv`.

Contents:

- `search_strategy_record.md`: submission-ready record with title, abstract, methods summary, databases, dates, and full search strings
- `search_strategy_record.txt`: plain-text version for copy-paste submission
- `searchrxiv_submission_checklist.md`: short checklist for the final manual deposit

This packet is meant to complement, not replace, the main public repository and Zenodo archive.
