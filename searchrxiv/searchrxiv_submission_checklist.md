# searchRxiv Submission Checklist

1. Create or sign in to the searchRxiv account used for the corresponding author.
2. Create a new search strategy record.
3. Use `search_strategy_record.md` or `search_strategy_record.txt` for the title, abstract, keywords, and query text.
4. Cross-link the final GitHub repository URL and the Zenodo DOI once available.
5. Record the minted searchRxiv DOI and paste it into `metadata/data_and_code_availability_statement_draft.md` if you want the manuscript to mention searchRxiv explicitly.
6. Keep the deposited search strings aligned with `release/supplement/Additional_file_1_Protocol_and_search_appendix.md`.
