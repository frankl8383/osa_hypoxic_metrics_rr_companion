# Environment Notes

The final submission package was generated in a local Linux environment with:

- Python `3.13.x`
- `Pillow`
- `pypdf`
- standard-library modules only beyond those packages for core script logic

Optional tools used for cross-render and compatibility QA:

- `LibreOffice` / `soffice`
- `pandoc`
- `Ghostscript`
- `poppler-utils`

The final export script expects the broader `osa_meta_20260323` workspace layout when regenerating all submission artifacts. This public package therefore serves primarily as a transparency deposit rather than a fully self-contained environment image.
